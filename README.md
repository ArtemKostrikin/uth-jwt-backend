# Auth JWT Backend

Бэкенд для аутентификации пользователей с JWT и MySQL.

## Функционал
- Регистрация пользователей (bcrypt)
- Авторизация с JWT токеном
- Валидация входных данных
- CORS support

## Стек
- Node.js / Express
- MySQL
- JWT (jsonwebtoken)
- bcrypt

## Запуск

```bash
npm install
node backend/index.js
